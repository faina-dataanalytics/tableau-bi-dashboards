# Mobile Game User Activity Analysis (Tableau)

## Projektbeschreibung
Dieses Projekt umfasst die Erstellung eines interaktiven Tableau-Dashboards zur Analyse der Nutzeraktivität in einem Mobile Game.
Der Fokus liegt auf User Engagement, Battle-Pass-Nutzung, durchschnittlicher Spielzeit sowie Unterschieden zwischen Altersgruppen.

Das Dashboard ist für Product Manager und Game Designer konzipiert und ermöglicht eine selbstständige Analyse ohne direkte Unterstützung durch Analyst:innen.

---

## Business-Ziele
- Bewertung der Nutzerbeteiligung an Battle-Pass-Aktivitäten
- Analyse der durchschnittlichen Spielzeit pro Nutzer
- Vergleich der Aktivität zwischen verschiedenen Altersgruppen
- Untersuchung zeitlicher Trends im Nutzerverhalten
- Bereitstellung eines Executive Dashboards mit interaktiven Filtern

---

## Tools & Methoden
- Tableau Desktop
- Calculated Fields
- LOD Expressions
- Date Functions (Monat, Quartal)
- Heatmaps
- Benutzerdefinierte Text-Labels (Zeitformat HH:MM)
- Interaktive Filter
